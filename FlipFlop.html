Interactive FlipFlop Circuit

This is a Circuit comprised of multiple logic gates which have been shown in the previous pages. It includes an OR Gate, AND Gate and NOT Gate.
The purpose of this circuit is to show how computers store memory. An input in the top slot results in a value stored in the output. The output can then be removed by using the bottom input which removes the stored input.
